## 🌍 What is Build Guild?

Build Guilds are **free, 1-day hardware meetups** running April 13–19 in cities all around the world,
organized as part of [Hack Club Blueprint](https://blueprint.hackclub.com/guilds).

Every guild is organized by a local Hack Clubber. Some have workshops, some have project demos,
some just build and hang out together — but all of them celebrate **hardware and community**.

**Build Guild Bharatpur** is Bharatpur's very own edition of this global celebration. 🇳🇵

## ❓ FAQ

**Do I need hardware experience?**
Nope! Build Guilds is open to complete beginners and experienced makers alike.

**Is it really free?**
Yes! Hack Club funds the event per attendee. You pay nothing.

**Do I need to be a Hack Club member?**
No. Anyone aged 13–18 can attend.

**What is Hack Club Blueprint?**
[Blueprint](https://blueprint.hackclub.com) is Hack Club's hardware program, made in collaboration with AMD,
where teens get funded to build real hardware projects.
